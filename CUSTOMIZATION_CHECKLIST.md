# Customization Checklist

Complete these steps to personalize your website with real data.

## 📝 Step 1: Update Channel Data

Open `/lib/channelData.ts` and update:

### Owner Information
- [ ] Change `owner.name` to your artist name
- [ ] Update `owner.role` with your title
- [ ] Write your bio in `owner.bio`
- [ ] Update `owner.image` path to your profile image

### Channel Statistics
- [ ] Update `channel.subscriberCount` with your YouTube subscribers
- [ ] Update `channel.videoCount` with number of videos
- [ ] Update `channel.totalViews` with total channel views
- [ ] Change `channel.youtubeUrl` to your YouTube channel URL

### Featured Videos
For each video in the `videos` array:
- [ ] Update video title
- [ ] Add video description
- [ ] Update view count
- [ ] Set upload date
- [ ] Set video duration
- [ ] Add thumbnail image path (or use default)
- [ ] (Optional) Add YouTube video ID for direct links

### Social Media Links
- [ ] Update YouTube URL
- [ ] Add Spotify artist link
- [ ] Add Instagram profile URL
- [ ] Add Twitter handle link
- [ ] Add TikTok profile link
- [ ] Add SoundCloud profile link

### Contact Information
- [ ] Update email address
- [ ] Add phone number (optional)
- [ ] Add location/city (optional)

## 🖼️ Step 2: Add Images

### Profile Image
- [ ] Create/prepare your profile image (1200x1200px, JPG)
- [ ] Save as `/public/artist-profile.jpg`
- [ ] Update `owner.image` path in channelData.ts

### Video Thumbnails
For each featured video:
- [ ] Prepare video thumbnail image (1280x720px, JPG)
- [ ] Save to `/public/` folder
- [ ] Update thumbnail path in video entry

### Optional: Additional Images
- [ ] Channel banner/header image
- [ ] Favicon (browser tab icon)
- [ ] Social media preview images

## 🔗 Step 3: Verify Links

- [ ] YouTube channel URL works
- [ ] All social media links are correct
- [ ] Email contact link is correct
- [ ] Video links point to correct videos (if using YouTube IDs)
- [ ] Newsletter subscription setup (if using email service)

## 🎨 Step 4: Styling (Optional)

### If you want to change colors:
- [ ] Open `/app/globals.css`
- [ ] Update color values in `:root` section
- [ ] Update corresponding `.dark` theme colors
- [ ] Test colors on different sections

### If you want to change fonts:
- [ ] Update `/app/layout.tsx` font imports
- [ ] Update font references in `/app/globals.css`
- [ ] Test on different devices

## 📱 Step 5: Test Responsiveness

- [ ] Test on mobile phone (portrait)
- [ ] Test on mobile phone (landscape)
- [ ] Test on tablet
- [ ] Test on desktop
- [ ] Check all links work properly
- [ ] Verify images load correctly
- [ ] Test navigation menu

## 🚀 Step 6: Deployment Preparation

- [ ] Update SEO metadata in `/app/layout.tsx`
- [ ] Add favicon to `/public/` folder
- [ ] Check for broken links
- [ ] Verify all external links open in new tabs
- [ ] Test contact forms (if configured)

## 📊 Step 7: Analytics & SEO

- [ ] Add Google Analytics (optional)
- [ ] Set up email notifications for newsletter
- [ ] Configure social sharing preview images
- [ ] Add robots.txt and sitemap.xml (auto-generated by Next.js)

## ✅ Final Checks Before Launch

- [ ] All channel data matches real statistics
- [ ] All images are high quality and correctly sized
- [ ] All links work and point to correct destinations
- [ ] Website looks good on all devices
- [ ] Navigation is intuitive
- [ ] Call-to-action buttons are clear
- [ ] Contact information is accurate
- [ ] Social links are all active
- [ ] Video information is current

3. [ ] Deploy to production
4. [ ] Test live website
